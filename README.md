# Tab Group Auto-Sleep

[![Release](https://img.shields.io/github/v/release/dragonish/Tab-Group-Auto-Sleep)](https://github.com/dragonish/Tab-Group-Auto-Sleep/releases/latest) [![GPL-3.0](https://img.shields.io/github/license/dragonish/Tab-Group-Auto-Sleep
)](/LICENSE)

When opening a tab group, automatically sleep other inactive tabs in the group.

## Installation

## Development

### Environment

- Install [Node.js](https://nodejs.org/) 22+

### Initialization

```bash
# Install pnpm
npm install -g pnpm

# Installation dependency
pnpm install
```

### Build

```bash
pnpm run build
```
